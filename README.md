# Hyprland Dotfiles

A collection of configuration files and scripts for a customized Hyprland desktop environment on Linux.

## Features

- Modular configuration for Hyprland and related tools
- Theming for GTK, Qt, terminal, and more
- Scripts for automation and system integration
- Pre-configured widgets, status bars, and utilities
